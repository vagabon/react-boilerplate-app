# React Boilerplate App

Librairie NPM : <a href="https://www.npmjs.com/package/@vagabond-inc/react-boilerplate-app" target="_black">react-boilerplate-app</a>

- modules :
  - auth (login, register, forget password)
  - admin (crud)
  - news (crud)
  - notification
  - contact page
  - user
- api :
  - service + hook
- app :
  - bouttons :
    - refresh
    - report
    - input
  - Chart (mui-x-chart)
  - Form (formik + yup générique)
  - image :
    - upload + download d'image de l'api
  - infinite scroll
  - hook :
    - scroll
    - resize
  - reducer :
    - Common Reducer
    - Base Reducer génénique (pour du CRUD)
  - template :
    - header
    - footer
    - menu / drawer
    - cookie consent
    - theme
    - i18n + sélection de la langue
